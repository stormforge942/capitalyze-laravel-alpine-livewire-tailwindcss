# Features:
- User can create comparison charts (tabs)
- User can switch between tabs

# Detail: 
Each tab contains:
- companies (string[])
- metrics (string[])
- data ({
    data: {
        [company]: {
            [period]: {
                [metric]: {
                    [date]: number
                }
            }
        }
    }
    dates: [...dates],
})
- filters ({
    period: annual | quarterly,
    dateRange: [min, max],
    unit: string,
    decimalPlaces: number
})
- config ({
    panel: string,
    metricAttributes: {
        [metric]: {
            show: boolean,
            chartType: line | bar
        }
    }
})


# challenges:
- Smoother tab switching
- No jankiness

# Approach
- Get the tabs to work first
- Then work on the part to sync the data with the tabs
- Then work on the UI to display the data dynamically
